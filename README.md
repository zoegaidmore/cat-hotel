# Kitty Paws

Kitty Paws is a responsive and accessible website for a cat hotel. The main idea of the project is to present a cozy hotel for cats called `Kitty Paws`, with dedicated pages for the home experience, rooms, contact information, about section, and customer reviews.

## Project Highlights

- Built with semantic HTML and CSS
- Uses BEM naming conventions
- Designed to be responsive across screen sizes
- Improved with accessibility-focused markup and interactions
